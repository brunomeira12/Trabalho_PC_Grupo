Sobre o Nosso Projeto

Nosso projeto na disciplina de Pensamento Computacional consiste em desenvolver um site usando HTML e CSS. O objetivo é aplicar o que aprendemos para criar uma página funcional e visualmente atrativa. Estamos focando na estruturação do conteúdo em HTML, com atenção à semântica e acessibilidade, enquanto usamos CSS para garantir um design moderno e responsivo. Esse projeto tem sido uma excelente oportunidade para consolidarmos nossos conhecimentos na prática.
